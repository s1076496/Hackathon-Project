# College Major Decider
## Built by Eli Ferency and Yassin Fathalla for the AP CSP Hackathon.

This small web app shows statistics for various college majors. All numeric data shown in the app is from 2012.

## Files

- `index.html` — main entry
- `majorList.html` — list of majors
- `majorStats.html` — major details & stats
- `script.js` — application logic
- `styles.css` — styles

## Run locally

Option 1 — Python (simple, no dependencies):

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000/index.html in your browser.

Option 2 — Node.js (`http-server`):

```bash
npx http-server
```

## Notes

- Data source: 
    -Numeric values in the app are from 2012.
